# The Demonstration Method

Renaming a file and checking our results.
>``` shell
>kali@kali:~$ ls *.txt
>
># ========== Expected Result ==========
>oldfilename.txt
># =====================================
>
>kali@kali:~$ mv oldfilename.txt newfilename.txt
>
>kali@kali:~$ ls *.txt
># ========== Expected Result ==========
>newfilename.txt
># =====================================
>```
