# WHOIS forward lookup for megacorpone.com
whois megacorpone.com -h <WHOIS_IP_ADDRESS>
```
```
# WHOIS reverse lookup for IP 38.100.193.70 
whois 38.100.193.70 -h <WHOIS_IP_ADDRESS>
